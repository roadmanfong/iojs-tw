npm 週報 #14

Cordova 套件轉移到 npm

現在有許多框架持續將套件轉移到 npm 的趨勢，Apache Cordova 宣布將其核心套件轉移到 npm，也鼓勵插件開發者這麼做。假如您是 Cordova 開發者，請直接閱讀開發指南。

CLI 消息

npm@next 這周帶來了 2.9.0 版本，此更新有兩個小修正，當您使用 npm outdated 或 npm update 指令更動本地端套件時，版本前面的前置名稱現在可以透過 npm 來設定了。

部署 github 頁面

您還在維護個人 Github 網站嗎？您可以透過 npm run scripts 指令讓部署過程更順暢。Jessica Lord 寫了一個簡單範例在 git-it 套件內。

io.js 週報 2015.04.10

io.js 1.6.4 release

這週我們有一個 io.js 釋出版本 v1.6.4，完整的更新項目可以在 GitHub 上找到。

主要更新

  • npm: npm 升級到 2.7.5，細節可以參閱 npm CHANGELOG.md。這個版本包含了兩個重要的安全性更新。
  • openssl: OpenSSL 1.0.2a #1325 (Shigeki Ohtsu) 前期準備工作已經完成。更多細節可以參考 #589。
  • timers: 未被參考 timers 所造成記憶體洩漏的問題已經被修復，包含一些 timers 相關的問題 #1330 (Fedor Indutny)。這些修復了在 #1075 回報的問題。
  • android: 現在已經可以編譯讓 Android 以及相關設備使用的 io.js 版本 #1307 (Giovanny Andres Gongora Granada)。

已知問題

  • 在 beforeExit 執行未參考的 timer 仍存在一些問題需要解決 #1264。
  • REPL 的代理配對會導致終端凍結 #690。
  • io.js 不可能編譯成靜態函式庫 #686。
  • process.send() 如同文件所建議並非同步，在 1.0.2 版新增了回歸，請見 #760 及修改 #774。
  • 當某個DNS查詢正在處理中，呼叫 dns.setServers() 會導致程序崩潰在某個失敗的斷言 #894。

社群更新

  • Node 基金會的開發策略草案在這邊。
  • io.js 支援 ARMv8 / ARM64 平台。
  • 持續著 node.js/io.js 新開發方針的工作。
  • 星期三的 TC 會議。

即將舉行的活動

  • JSConf 烏拉圭的門票已經開賣，時間是 4/24~4/25 於烏拉圭蒙特維多。
  • NodeConf Adventure 則是 6/11-14 於加州沃克溪農場。
  • CascadiaJS 門票開賣，時間是 7/8~7/10 於華盛頓州。
  • NodeConf EU 門票開賣，時間是 9/6~9/9 於愛爾蘭瓦特福。
  • nodeSchool 東京 將會在 4/12 於日本東京舉辦。
